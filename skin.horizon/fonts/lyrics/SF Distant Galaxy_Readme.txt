SF Distant Galaxy - Version 2.5 - Instructions
------------------------------------------------------------------------

 - Connecting Letters

   - To connect any two characters use SF Distant Galaxy Alternate or
     SF Distant Galaxy AltOutline.  The first letter to be connected
     should be a lowercase and the second should be an uppercase.  For
     example, type the following line using either SF Distant Galaxy or
     SF Distant Galaxy AltOutline:

     sTaR wArS

     (note: SF Distant Galaxy and SF Distant Galaxy Outline characters
     do not connect and are designed to work in conjunction with SF
     Distant Galaxy Alternate and SF Distant Galaxy AltOutline)

 - Adding Extensions Using SF Distant Galaxy Alternate

   - To add extensions using SF Distant Galaxy Alternate, press < for
     bottom extensions and > for top extensions.  For example, type the
     following line using SF Distant Galaxy Alternate:

     >>TiE fIgHtEr<<

     (note: SF Distant Galaxy does not have extensions)

 - Adding Extensions Using SF Distant Galaxy AltOutline

   - To add extensions using SF Distant Galaxy AltOutline, press < for
     an open bottom extension, > for an open top extension, [ for a
     closed bottom right extension, ] for a closed bottom left
     extension, { for a closed top right extension, and } for a closed
     top left extension.  For example, type the following line using SF
     Distant Galaxy AltOutline:

     {>>TiE fIgHtEr<<]

     (note: SF Distant Galaxy Outline does not have extensions)

 - Adding Star Wars Title Bar

   - To add a Star Wars title bar using SF Distant Galaxy Symbols,
     press $ in the middle of any word or phrase.  For example, type
     the following line using SF Distant Galaxy Symbols:

     aure$besh
